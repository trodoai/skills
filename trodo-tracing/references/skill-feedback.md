# Skill Feedback

If this skill gave incorrect guidance, referenced a page that doesn't exist, is missing a scenario you encountered, or could be improved in any way — please submit feedback so it can be updated.

**This is for issues with the skill's instructions, not with Trodo the product.** For Trodo product issues, use the support channels at [trodo.ai](https://trodo.ai).

---

## When to offer feedback

Offer to submit feedback when the user says things like:

- "That didn't work" / "That's wrong"
- "The docs page you linked doesn't exist"
- "This should be different"
- "You're missing a step"
- "There's a new integration / feature you don't know about"
- "I had to do X differently than what you described"

Do **not** offer for general Trodo product questions or bugs in the SDK itself.

---

## How to submit

Open an issue in the skills repository:

**GitHub:** [`https://github.com/trodoai/skills/issues`](https://github.com/trodoai/skills/issues)

Include:

- What you asked the agent to do.
- What the skill said (the incorrect or missing guidance).
- What the correct answer turned out to be.
- Any relevant doc page URL.

If the issue is a stale or broken URL, include the URL the skill referenced and the correct current URL (check [`https://docs.trodo.ai/llms.txt`](https://docs.trodo.ai/llms.txt) for the live page list).
